This type of card is usually called a dynamic achievement card, streak card, or progress badge card. The image you shared combines four elements:

User progress data (Day 5 completed)

Pre-designed visual template (shield, colors, branding)

Automated image generation

WhatsApp delivery

You can build this for your community without manually designing each card.

How the system works
User completes activity
        ↓
Database updates progress
        ↓
Automation workflow triggers
        ↓
Dynamic card generated
        ↓
Image sent automatically on WhatsApp
Step 1: Create the card template
Design the card once using:

Canva

Figma

Adobe Express

Keep variable elements as placeholders:

{user_name}

{day_number}

{achievement_text}

{streak_count}

{profile_photo}

{next_day_status}

Example:

DAY {day_number}

Today felt {mood_text}

Achieved by:
{user_name}
Export the background without dynamic text.

Step 2: Store community data
You need a simple database containing:

User ID	Name	Phone	Current Day	Status	Last Activity
Tools:

Google Sheets (small communities)

Airtable

Notion

Firebase

Supabase

Step 3: Generate images automatically
No-code approach
Use:

Bannerbear

Placid

APITemplate.io

These tools allow you to:

Upload a template

Define dynamic fields

Generate images through APIs

Example API request:

{
  "user_name": "Chitwan Arora",
  "day_number": "5",
  "achievement_text": "Today felt light, calm, and refreshing"
}
The service returns an image URL automatically.

Custom-code approach
If you need complete control:

Node.js + HTML/CSS + Puppeteer

Python + Pillow

Python + HTML templates + Playwright

Workflow:

Database → Python Script → Generate PNG → Store Image
Best for:

Large communities

Custom animations

Lower long-term cost
